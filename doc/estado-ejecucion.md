## 📝 Formato de Entrada al Historial

Cada prompt debe registrar sus acciones usando el siguiente formato:

```
### [YYYY-MM-DD HH:MM] — [TIPO]: Descripción breve
- **Acción:** Descripción detallada de lo que se hizo
- **Archivos creados/modificados:** lista de archivos
- **Resultado:** Exitoso / Fallido / Parcial
- **Observaciones:** Notas adicionales, errores encontrados, decisiones tomadas
```

**Tipos de entrada:**
- `INICIO` — El prompt comienza a ejecutar la fase
- `ACCIÓN` — Paso intermedio significativo dentro de la fase
- `COMPLETADO` — La fase finalizó satisfactoriamente
- `ERROR` — Se encontró un problema (incluir descripción y solución)
- `BLOQUEADO` — La fase no puede continuar (incluir motivo)
